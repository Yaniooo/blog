这篇文章用途用于：记录文章文法、记录外挂标签写法以及记录任何在建立博客时需要记录的东西

博客创建

安装Node.js

Node.js用于本地编译生成静态网页
安装教程 https://m.runoob.com/nodejs/nodejs-install-setup.html
安装包下载链接 https://nodejs.org/en/download/
到下载地址选择对应版本下载安装，根据自己的情况安装和配置，注意环境变量和安装位置

安装git

git用于将本地数据提交到Gitee服务器
安装教程 https://m.runoob.com/git/git-install-setup.html
安装包链接 http://git-scm.com/downloads
到下载地址选择对应版本下载安装，根据自己的情况安装和配置，注意安装位置

配置Gitee

注册登录并新建一个Gitee仓库 https://gitee.com/
开启Gitee Pages服务，记录网站地址

安装Hexo

打开Node.js并输入npm install -g hexo-cli 下载完毕之后可输入hexo v查看版本信息

初始化Hexo

打开Node.js，选择本地hexo数据目录输入hexo init "博客文档路径"，cd "博客本地路径"，npm install
输入cd指令后显示出来的路径不改变，直接进行下一步操作即可
完成以上步骤之后，博客文件夹里有如下文件

浏览本地博客

在上一步博客根目录里鼠标右键Git Bash打开窗口，输入hexo s启动本地服务，快捷键Ctrl+C停止服务
进入localhost:4000页面查看博客

博客上传到Gitee

配置Git全局信息

博客根目录里鼠标右键Git Bash打开窗口输入以下命令
git config --global user.name "你的Gitee用户名"
git config --global user.email "你Gitee使用的邮箱"

修改配置文件

博客根目录里找到_config.yml文件
设置url和root信息为：
url: http://yanio.gitee.io/myblog #Gitee Pages所得到的链接
root: /MyBlog/ #Gitee里的博客仓库
